# DevOps Plugin

CI/CD, Docker, and deployment automation.

## Features

### Commands
| Command | Description |
|---------|-------------|
| `/ci setup` | Set up CI/CD pipeline |
| `/ci update` | Update existing pipeline |

### Agent
DevOps Engineer with expertise in:
- GitHub Actions, GitLab CI
- Docker and Kubernetes
- Infrastructure as Code
- Cloud platforms

### Skills
- CI/CD pipeline patterns
- Docker best practices
- Kubernetes deployments

## Installation

```bash
/plugin install devops@xalapm-marketplace
```
