# Code Review Plugin

Thorough code review with actionable feedback.

## Features

### Commands
| Command | Description |
|---------|-------------|
| `/review` | Review code for quality |
| `/review target=file.ts` | Review specific file |
| `/review focus=security` | Focus on security |

### Agent
Code Review Agent with expertise in:
- Clean code principles
- SOLID patterns
- TypeScript best practices
- React patterns
- Security considerations

### Skills
- Issue identification
- Refactoring suggestions
- PR review templates
