# Documentation Plugin

Generate and maintain project documentation.

## Features

### Commands
| Command | Description |
|---------|-------------|
| `/docs readme` | Generate/update README |
| `/docs api` | Generate API documentation |
| `/docs changelog` | Update changelog |

### Agent
Documentation Writer with expertise in:
- README and project docs
- API documentation (OpenAPI, TypeDoc)
- Code comments (JSDoc)
- Changelog management

### Skills
- README patterns and badges
- JSDoc/TSDoc conventions
- Changelog best practices

## Installation

```bash
/plugin install documentation@xalapm-marketplace
```
