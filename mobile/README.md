# Mobile Development Plugin

Cross-platform mobile development with Expo and React Native.

## Features

### Commands
| Command | Description |
|---------|-------------|
| `/screen create` | Create a new mobile screen |

### Agent
Mobile Development Agent with expertise in:
- Expo SDK and managed workflow
- React Native core
- iOS and Android specifics
- React Navigation
- Native modules
- App deployment

### Skills
- Expo/React Native patterns
- Authentication and secure storage
- Push notifications
- Gestures and animations
- Offline support

## Installation

```bash
/plugin install mobile@xalapm-marketplace
```
